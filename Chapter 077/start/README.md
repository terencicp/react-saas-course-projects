# Chapter 077 — TanStack Query on optimistic comments (starter)

This is the starting code repo for the chapter 077 project of the React SaaS course.

This repo builds on the previous project: chapter 062 (the production invoice list-view surface).

You bolt a polling, infinite-scrolling, optimistically-added comment thread onto the Ch062 invoice detail page, with TanStack Query scoped to a single client leaf.

The provided base (Ch062 invoices surface, the in-memory store with seeded comments, the inspector, and the read/write seam wrappers) is complete. Search for the work with `rg "TODO\(L" .` — every student-owned file carries a `// TODO(L<n>)` marker.
