# Chapter 028 — Themed product surface (starter)

This is the starting code repo for the chapter 028 project of the React SaaS course.

This is the root project of the course: it is built from scratch and does not build on any previous project.

Stack: Next.js 16 (App Router, `src/` layout), React 19, TypeScript, Tailwind v4, shadcn/ui, next-themes. Tooling: pnpm, Biome, Vitest.

Run `pnpm install`, then `pnpm dev`. The student-owned component and hook bodies are stubbed with `// TODO(L<n>)` markers; run `rg "TODO\(L" src` to enumerate the work.
